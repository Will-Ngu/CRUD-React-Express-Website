# Installation Instructions (Docker)

Note: These instructions assume you have Docker and docker-compose installed on your machine.
1. Clone this repository
2. Within the root directory of the cloned repository, run ```docker-compose up``` (Note that this may take a long time due to installing npm dependencies)
3. Once the docker compose script finishes initializing, you will be able to use the application at ```localhost:3000```

# Installation Instructions (Not Docker)
1. Clone this repository
2. Within the root directory of the cloned repository, run ```npm start``` (Note that this may take a long time due to installing npm dependencies)
3. Once the script has finished starting up, you will be able to use the application at ```localhost:3000```
